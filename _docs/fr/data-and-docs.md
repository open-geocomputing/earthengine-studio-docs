## Recherche de données

La recherche locale combine le catalogue Google Earth Engine et Awesome GEE Community Datasets. Elle classe les identifiants, titres, mots-clés, bandes et fournisseurs, tout en maintenant une diversité de sources.

Un résultat officiel peut charger les métadonnées STAC détaillées: bandes, unités, résolution, propriétés, masques et conditions. Les entrées communautaires conservent leur attribution et leurs liens source. Vérifiez toujours le [catalogue Earth Engine](https://developers.google.com/earth-engine/datasets/catalog?hl=fr) pour le statut officiel.

## Panneau Docs

**Docs** rassemble le registre Earth Engine, les signatures supplémentaires, les extensions Studio, les cartes et les exportations. Chaque entrée indique si elle vient de Google ou de Studio. La même source alimente la complétion Monaco. Utilisez la [référence API Google](https://developers.google.com/earth-engine/apidocs?hl=fr) pour la documentation complète de la plateforme.
